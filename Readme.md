# Dotfiles Repo


## Requirements

### NvChad install
```bash
git clone https://github.com/NvChad/starter ~/.config/nvim && nvim
```

### FiraMono Nerd Font
https://www.nerdfonts.com/font-downloads

### cargo binaries
```bash
cargo install \
atuin \
just \
pueue \
fd-find \
du-dust \
cargo-update \
flamegraph \
ripgrep \
starship \
tokei \
tealdeer \
zoxide
```

### Clipboard integration
```bash
sudo apt install wl-clipboard
```
